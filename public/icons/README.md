# アイコンファイル

このディレクトリには以下のアイコンファイルが必要です：

- `icon16.png` - 16x16ピクセル
- `icon48.png` - 48x48ピクセル
- `icon128.png` - 128x128ピクセル

## 作成方法

1. デザインツール（Figma、Photoshopなど）でアイコンを作成
2. PNG形式で各サイズをエクスポート
3. このディレクトリに配置

## デザインガイドライン

- シンプルで認識しやすいデザイン
- Chrome拡張機能のアイコンガイドラインに準拠
- 透過背景を使用
- 視認性の高い色使い

## 一時的な対応

現在は開発中のため、アイコンファイルは後で追加します。
ビルド時にエラーが出ないよう、プレースホルダー画像を用意してください。
